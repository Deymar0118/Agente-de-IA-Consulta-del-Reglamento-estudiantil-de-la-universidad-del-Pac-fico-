# Agente-de-IA-Consulta-del-Reglamento-estudiantil-de-la-universidad-del-Pac-fico-
Agente de Inteligencia Artificial para responder consultas sobre la reglamentación de la Universidad del Pacífico.

Planificicacion
•	Nombre: Asistente Inteligente para Consultas sobre la Reglamentación de la Universidad del Pacífico
•	Objetivo: Crear un agente de AI que dé respuesta a consultas relacionadas con la reglamentación universitaria en la Universidad del Pacífico.
•	Tareas
	Tema del agente: Reglamento Estudiantil
	Problema que resuelve:  Los estudiantes y/o directivos de la universidad el Pacífico deben hacer consultas y acuerdos institucionales para dar paso a procesos académicos y administrativos. Con este agente de IA, se le dará una respuesta adecuada sin requerir de mucho tiempo
	Usuarios: Estudiantes, Docentes y Administrativos de la Universidad del Pacífico.
	Documento: PDF
